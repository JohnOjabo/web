      The good thing about Themeforest is that they have Envato WordPress Toolkit plugin (free plugin installed separately) which helps you to automate updates just like you could with free themes from WordPress.org.

      Also if you are doing code changes to themes then you need to create a Child Theme regardless if this theme comes from WordPress.org, Themeforest or any other theme store. Otherwise you will always lose those code customizations once you update and there is no simpler way than to make changes via Child Themes and then update Parent Theme any time a new version comes out.

      I hope that cleared things up for you as updates works the same for any WordPress theme but at least WordPress.org and Themeforest has some nice tools for it while others don’t with some very few exceptions.

Best Masonry Grid Style WordPress Themes

Note: Masonry is a revolutionary grid layout plugin for jQuery. Consider the flip side of CSS floats to have a clearer view of how masonry layout works.

While CSS floats push elements to the left or right, masonry moves elements vertically. It positions each element in the next open spot in the grid, minimizing vertical gaps between items of varying height.

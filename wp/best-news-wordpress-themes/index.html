      Hi!

      None of comments are removed but all are withheld for moderation therefore doesn’t appear right away.

      Here are answers to your questions:

      1. WordPress is slow on its own and it loads slow even when you are using a starter themes such as Underscores. WordPress is database heavy application and it takes all information from database and it takes a lot of time. However, you can speed things up by using cache. Caching plugins will make sure to cache content and serve it from disk without trying to connect to database each time. This will speed things up significantly.

      It’s not an ideal solution because if your server is slow then even cached content won’t help much. It will load faster but never as fast as Colorlib or other well optimized WordPress websites. For that you need a good hosting which is built specially for WordPress. For more information about hosting you can read this guide.

      2. Custom framework means that they have their own Theme Options panel, layout settings and other features. However, this has nothing to do with WordPress plugins. Theme doesn’t affect plugins and you can use whatever plugin you want. Some might not work but that’s the same with any other theme and those are very few unpopular plugins. That’s usually plugin’s fault and has nothing to do with theme.

      3. Images are optional. You can not obligated to use them. I love images and I don’t like reading long text blocks but it’s about your personal preference. Theme will work perfectly fine without images. Of course it will look slightly different but that’s a minor thing.

      Let me know if you have any other questions about WordPress themes, hosting or platform as a whole.

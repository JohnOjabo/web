Pre-purchase FAQ

1. I want to make multiple websites with the same theme – how many licenses do I need to buy?

One license is allowed per one public domain only. If you want to create a second website from the same theme, you’ll need to purchase another license. By purchasing a theme again you’re purchasing a new license. The process of purchasing another license is the same as purchasing a theme for the first time. Please read this article for more information about regular license.

2. Can I use the theme on a development/staging/test site, then move it to a live site when finished?

Yes. You can use the same purchase code to activate the theme on both development and live websites. Once the site migration is successful, please disable the license and delete the theme. It is not allowed to have the theme installed in two places permanently using the same purchase code.

3. Can I trial test the theme before I purchase it?

Unfortunately, trial testing is not available. However, if you want to see how the theme works and check out the demos, please visit the live preview website. If you want to take a closer look under the hood, the theme documentation includes screenshots and detailed info.

If you have any pre-sale questions please use the comments area or the live chat widget with our live preview website and ask us anything.

4. Does the theme support any other page builders?

The theme has been designed specifically to work with WPBakery Page Builder. The theme custom elements and pre-built blocks require this plugin to work. There are no other page builders compatible with this theme.

5. Does the theme provide full RTL support?

The theme is not optimized to be used on RTL websites. All custom elements and blocks are optimized to be used only on LTR websites.

6. Is it compatible with the WordPress Multisite?

Yes. You can build a multi-site network and use the Sway theme. This feature has been extensively tested and there are no known issues.

7. Can I translate the theme?

Yes. All text strings have been internationalized and can be translated to your local language. If you want to publish your content in multiple languages, the theme is compatible with WPML & Ploylang plugins.

8. How long I will get updates for this theme?

As long as the theme is available on ThemeForest, you will be able to get unlimited free updates.

9. Does the theme package include any PSD files?

The PSD files can be purchased separately from here.
